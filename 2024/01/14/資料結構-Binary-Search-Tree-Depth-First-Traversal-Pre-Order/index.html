紀錄 Binary Search Tree 的 Depth First Traversal 在 Pre order 的狀態下是怎麼遍歷的

pre-order 的遍歷特性

這邊說明一下 pre-order 的遍歷特性是
先觸碰 parent node ，
接著，先將左邊分支的 node 全部碰完，
最後，再跑右邊分支去碰右邊分支上的 node。

觸碰順序如下圖的小圓圈數字的順序所示

pre-order 遍歷的方式很適合在我們想要複製整個 tree 的時候使用。
